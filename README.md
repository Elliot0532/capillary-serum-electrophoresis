# capillary-serum-electrophoresis
File Structure
0830_DesNet.ipynb: Contains code for the DesNet algorithm
0830_MobileNet_0.938.ipynb: Contains code for the MobileNet algorithm with an accuracy score of 0.938
0830_ResNet.ipynb: Contains code for the ResNet algorithm
0731Vgg16(val_0_918).ipynb: Contains code for the VGG-16 algorithm with a validation accuracy of 0.918
0801html.py: A small program for machine recognition in clinical settings

How to Use
Clone this repository to your local machine
git clone https://github.com/yourusername/yourrepositoryname.git
Open the .ipynb files in a Jupyter Notebook environment for algorithm testing and debugging
Run 0801html.py to use the machine recognition tool in a clinical setting
